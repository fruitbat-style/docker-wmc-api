# Project Rules

- Never add Co-Authored-By to git commit messages.

## Pull Request Format

When creating PRs, always use this exact format:

- **Title**: Short, under 70 characters
- **Body**:

```
## Summary
- Bulleted list of concrete changes (what was added/changed/removed)

## Test plan
- [ ] Checkbox items for manual verification steps
```

- No extra sections beyond Summary and Test plan
- No emojis
- No "Generated with Claude Code" footer
